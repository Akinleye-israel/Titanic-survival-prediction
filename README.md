🚢 Titanic Survival Prediction
📌 Project Overview

This project is a supervised machine learning task that predicts whether a passenger survived the Titanic disaster. It uses passenger data such as age, gender, class, and fare to build and evaluate classification models.

The project demonstrates key machine learning steps including data preprocessing, exploratory data analysis, model building, and performance evaluation.

🎯 Objectives
Analyze and understand the Titanic dataset
Clean and preprocess the data
Build multiple classification models
Compare model performance
Identify the best model for prediction
